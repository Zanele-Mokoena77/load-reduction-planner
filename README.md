# load-reduction-planner
Load-Reduction Smart Planner is a command-line tool designed to help users plan their day around scheduled power outages.

# Problem Statement

In South Africa, load-shedding schedules are often difficult to interpret and plan around.
People need a simple way to understand when power will be off and when it is safe to perform important tasks.

# Solution

This application:

Loads load-shedding schedules from a local data source

Matches suburb and date inputs

Calculates outage periods

Suggests safe time windows for daily activities

The tool is intentionally built as a CLI to prioritise correctness, clarity, and testability.
